# 🪟 Windows 10 Desktop Simulator

A fully functional **Windows 10 Desktop** built with pure **HTML, CSS & JavaScript** — no frameworks, no libraries, just vanilla web tech!

![Windows 10 Desktop](https://wallpaperaccess.com/full/1124736.jpg)

---

## ✨ Features

### 🖥️ Desktop
- **Windows 10 Hero Wallpaper** as background
- **Draggable & Resizable** windows
- **Double-click** icons to open apps
- **Right-click** context menu on desktop
- **Live Clock & Date** in taskbar system tray

### 📌 Taskbar
- **Start Menu** with user profile (Bhatti)
- **Open app buttons** appear dynamically
- **Show Desktop** button (bottom-right)
- **System Tray** with clock, volume & network icons

### 🗂️ 8 Built-in Apps

| App | Description |
|-----|-------------|
| 🌐 **Google Chrome** | Fully working browser with address bar, bookmarks & iframe loading |
| 🔵 **Microsoft Edge** | Edge-style browser with Bing search & news cards |
| 📁 **File Explorer** | Sidebar navigation + file grid with folder/file icons |
| ⚙️ **Control Panel** | 16 system settings icons |
| 🗒️ **Notepad** | Type, copy, paste & download `.txt` files |
| 🖩 **Calculator** | Fully working calculator (+, −, ×, ÷, %, ±) |
| 🎵 **Media Player** | Animated visualizer bars, playlist & play/pause controls |
| 🗑️ **Recycle Bin** | Empty recycle bin display |

### 🪟 Window Controls
- **Minimize** — hides window, shows in taskbar
- **Maximize / Restore** — fullscreen toggle (also double-click title bar)
- **Close** — removes window & taskbar button
- **Drag** — move windows anywhere on screen
- **Resize** — drag bottom-right corner

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/windows10-desktop-simulator.git
cd windows10-desktop-simulator
```

### 2. Open in Browser
Just open `index.html` in any modern browser — **no server needed!**

```bash
# Option 1: Direct open
open index.html

# Option 2: Use VS Code Live Server
# Right-click index.html → "Open with Live Server"
```
